# Weather App

## Description

This weather application is a modular Flutter application designed to fetch and display real-time meteorological data from a public API.

## Tech Stack

- **Frontend:** Flutter
- **State Management:** Riverpod to manage loading, success, and error states
- **Data Handling:** HTTP for requests and JSON serialization/deserialization for model mapping

## Getting Started

To run this project locally, follow these steps

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/dilawarzAlgorithm/weather.git
    ```
2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Run**
    ```bash
    flutter run
    ```
